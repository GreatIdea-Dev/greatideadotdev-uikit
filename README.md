# Great Idea Development

Built with NextJS, TypeScript, and TailwindCSS

## Tech Stack and Features

- NextJS
- TypeScript
- TailwindCSS
- Multiple Themes and Color Schemes (user selectable)
- Dark Mode
- Light Mode
- Cursor Spotlight feature for text and card container
